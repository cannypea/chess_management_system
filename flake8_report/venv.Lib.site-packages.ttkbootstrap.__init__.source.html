1 """ttkbootstrap - A supercharged theme extension for tkinter.
2  
3 A modern flat style theme engine for tkinter that enables on-demand styling
4 of ttk widgets with over a dozen built-in themes inspired by Bootstrap.
5  
6 This package provides:
7 - A comprehensive collection of modern, flat-style themes
8 - Custom widgets extending tkinter/ttk functionality
9 - Easy-to-use styling API with color keywords
10 - Window and Toplevel classes with enhanced functionality
11 - Cross-platform compatibility
12  
13 Example:
14 ```python
15 import ttkbootstrap as ttk
16 from ttkbootstrap.constants import *
17  
18 # Create a themed window
19 root = ttk.Window(themename="darkly")
20  
21 # Create styled widgets
22 btn = ttk.Button(root, text="Click Me", bootstyle="success")
23 btn.pack(padx=10, pady=10)
24  
25 root.mainloop()
26 ```
27  
28 For more information, see: https://ttkbootstrap.readthedocs.io/
29 """
