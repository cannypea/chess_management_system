1 """Screen color picker (dropper) dialog for ttkbootstrap.
2  
3 This module provides a color dropper tool that allows users to select colors
4 directly from anywhere on the screen. It captures a screenshot and displays
5 a magnified view to help with precise color selection.
6  
7 Classes:
8 ColorDropperDialog: Interactive screen color picker widget
9 ColorChoice: Named tuple containing selected color in multiple formats
10  
11 Features:
12 - Click anywhere on screen to select color
13 - Magnified zoom window for precise selection
14 - Mouse wheel zoom in/out control
15 - Returns color in RGB, HSL, and HEX formats
16 - Cross-platform support (Windows and Linux only)
17  
18 Platform Support:
19 - Windows: Fully supported
20 - Linux: Fully supported
21 - macOS: NOT SUPPORTED (ImageGrab limitation)
22  
23 Known Issues:
24 High-DPI displays may require application to run in high-DPI mode for
25 accurate color selection. On Windows, this is handled automatically.
26 See: https://stackoverflow.com/questions/25467288
27  
28 Example:
29 ```python
30 from ttkbootstrap.dialogs.colordropper import ColorDropperDialog
31  
32 # Create color dropper

34 dropper.show()
35  
36 # Get selected color
37 color = dropper.result.get()
